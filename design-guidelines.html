Transition Moments

The signature motif from the brand deck. Inline SVG quadratic bezier arc — reliable in all browsers. Use at the bottom of green or dark sections where it "bites" into the cream below. 2–3 uses max across the full page.

04 — Visual World

Fair Elements on Web

Every fair element has a CSS/SVG web translation. These are the actual implementations — not references. Use them as motion objects, decorative layers, and ambient atmosphere.

Gradient orbs
Gradient Orbs
Coral+Teal, Teal+Lime. Absolutely positioned, blur(40–60px), opacity 0.4–0.7. Always z-index 0 behind content. Slow parallax on scroll (translateY 0.3× scroll delta).

Pinwheel
CSS conic-gradient on a circle element, clipped to an 8-petal shape. Rotation speed driven by scroll velocity via JS — fast scroll = fast spin. Delightful without demanding attention.
conic-gradient(colors) · clip-path: polygon · rotate(scrollVelocity × 0.8deg)
Tassel Bunting Garland
Inline SVG triangle flags on a curved path string. Full-width across section tops. Gentle ±3deg sway animation (3s ease-in-out infinite). All 5 brand colours across the flags in sequence.

ReactBits FluidGlass · R3F · useFBO · MeshTransmissionMaterial
Fluid Glass Bubbles
Built with ReactBits FluidGlass. A React Three Fiber Canvas uses useFBO to render a live frame buffer of everything behind the glass; a 3D GLB mesh applies MeshTransmissionMaterial with ior, thickness, chromaticAberration, and anisotropy — physically refracting that buffer for true, living distortion. Three modes: lens (sphere GLB), bar (capsule GLB), cube. Pointer-following via easing.damp3. VIRGIO hero uses mode="lens", ambient only.

/* eslint-disable react/no-unknown-property */
import * as THREE from 'three';
import { useRef, useState, useEffect, memo } from 'react';
import { Canvas, createPortal, useFrame, useThree } from '@react-three/fiber';
import {
  useFBO,
  useGLTF,
  useScroll,
  Image,
  Scroll,
  Preload,
  ScrollControls,
  MeshTransmissionMaterial,
  Text
} from '@react-three/drei';
import { easing } from 'maath';

export default function FluidGlass({ mode = 'lens', lensProps = {}, barProps = {}, cubeProps = {} }) {
  const Wrapper = mode === 'bar' ? Bar : mode === 'cube' ? Cube : Lens;
  const rawOverrides = mode === 'bar' ? barProps : mode === 'cube' ? cubeProps : lensProps;

  const {
    navItems = [
      { label: 'Home', link: '' },
      { label: 'About', link: '' },
      { label: 'Contact', link: '' }
    ],
    ...modeProps
  } = rawOverrides;

  return (
    <Canvas camera={{ position: [0, 0, 20], fov: 15 }} gl={{ alpha: true }}>
      <ScrollControls damping={0.2} pages={3} distance={0.4}>
        {mode === 'bar' && <NavItems items={navItems} />}
        <Wrapper modeProps={modeProps}>
          <Scroll>
            <Typography />
            <Images />
          </Scroll>
          <Scroll html />
          <Preload />
        </Wrapper>
      </ScrollControls>
    </Canvas>
  );
}

const ModeWrapper = memo(function ModeWrapper({
  children,
  glb,
  geometryKey,
  lockToBottom = false,
  followPointer = true,
  modeProps = {},
  ...props
}) {
  const ref = useRef();
  const { nodes } = useGLTF(glb);
  const buffer = useFBO();
  const { viewport: vp } = useThree();
  const [scene] = useState(() => new THREE.Scene());
  const geoWidthRef = useRef(1);

  useEffect(() => {
    const geo = nodes[geometryKey]?.geometry;
    geo.computeBoundingBox();
    geoWidthRef.current = geo.boundingBox.max.x - geo.boundingBox.min.x || 1;
  }, [nodes, geometryKey]);

  useFrame((state, delta) => {
    const { gl, viewport, pointer, camera } = state;
    const v = viewport.getCurrentViewport(camera, [0, 0, 15]);

    const destX = followPointer ? (pointer.x * v.width) / 2 : 0;
    const destY = lockToBottom
      ? -v.height / 2 + 0.2
      : followPointer ? (pointer.y * v.height) / 2 : 0;
    easing.damp3(ref.current.position, [destX, destY, 15], 0.15, delta);

    if (modeProps.scale == null) {
      const maxWorld = v.width * 0.9;
      const desired = maxWorld / geoWidthRef.current;
      ref.current.scale.setScalar(Math.min(0.15, desired));
    }

    gl.setRenderTarget(buffer);
    gl.render(scene, camera);
    gl.setRenderTarget(null);
  });

  const { scale, ior, thickness, anisotropy, chromaticAberration, ...extraMat } = modeProps;

  return (
    <>
      {createPortal(children, scene)}
      <mesh scale={[vp.width, vp.height, 1]}>
        <planeGeometry />
        <meshBasicMaterial map={buffer.texture} transparent />
      </mesh>
      <mesh
        ref={ref}
        scale={scale ?? 0.15}
        rotation-x={Math.PI / 2}
        geometry={nodes[geometryKey]?.geometry}
        {...props}
      >
        <MeshTransmissionMaterial
          buffer={buffer.texture}
          ior={ior ?? 1.15}
          thickness={thickness ?? 5}
          anisotropy={anisotropy ?? 0.01}
          chromaticAberration={chromaticAberration ?? 0.1}
          {...extraMat}
        />
      </mesh>
    </>
  );
});

function Lens({ modeProps, ...p }) {
  return (
    <ModeWrapper
      glb="/assets/3d/lens.glb"
      geometryKey="Cylinder"
      followPointer
      modeProps={modeProps}
      {...p}
    />
  );
}

function Cube({ modeProps, ...p }) {
  return (
    <ModeWrapper
      glb="/assets/3d/cube.glb"
      geometryKey="Cube"
      followPointer
      modeProps={modeProps}
      {...p}
    />
  );
}

function Bar({ modeProps = {}, ...p }) {
  const defaultMat = {
    transmission: 1,
    roughness: 0,
    thickness: 10,
    ior: 1.15,
    color: '#ffffff',
    attenuationColor: '#ffffff',
    attenuationDistance: 0.25
  };

  return (
    <ModeWrapper
      glb="/assets/3d/bar.glb"
      geometryKey="Cube"
      lockToBottom
      followPointer={false}
      modeProps={{ ...defaultMat, ...modeProps }}
      {...p}
    />
  );
}

Never more than 2–3 elements visible simultaneously. Fair elements are the seasoning — photography and typography are the meal. Each section should have one "resident" fair element maximum, used at low opacity or as ambient motion.

05 — Motion & Animation

How Things Move

Spring physics, not linear. Every motion should feel like the fair — organic, alive, never mechanical. Ambient motions barely register; triggered animations earn attention.

🌀
Pinwheel — Scroll Velocity
Rotation speed tied to scroll velocity. Fast scroll = fast spin, idle scroll = slow drift. Never completely stops — always a gentle 0.1 RPM ambient rotation.
rotate(scrollVelocity × 0.8deg) · ambient: 0.1rpm · easing: spring(1, 80, 10, 0)
🫧
Bubbles — Float & Fade
3–4 bubbles drift upward from below the hero. Random size (24–56px), random x-position, staggered start times. Loop infinite. Respect prefers-reduced-motion: paused state.
translateY(-110vh) · 10–14s linear infinite · opacity 0→0.7→0 at 10%/50%/90%
🎊
Bunting — Gentle Sway
The garland sways on a slow pendulum — ±3deg, 3s ease-in-out infinite. If you can consciously notice it's moving, it's too fast. It should feel like something you discover, not something that announces itself.
rotate(-3deg) ↔ rotate(3deg) · 3s ease-in-out infinite alternate · transform-origin: center top
💰
Price Reveal — The Money Moment
The most important animation on the page. Sequence: (1) retail price appears, (2) strikethrough draws across it (300ms), (3) "Cost to Make" label fades in (200ms), (4) new price slides up from below (400ms spring). Only triggers once on scroll entry.
Sequence: appear → stroke-dashoffset 300ms → label fade 200ms → price spring 400ms
⏱️
Countdown — Slide Transition
Each digit updates with a quick slide-down exit, slide-up entry (AnimatePresence). Clean, not "spinning drum." Individual digits animate independently so "12:47:03 → 12:47:04" only moves the last number.
exit: y:-16 opacity:0 · enter: y:16→0 opacity:0→1 · 150ms spring · digit-level isolation
📜
Scroll Entry — Fade + Rise
Every content block enters via fade + upward movement. Never from sides. Stagger children by 80ms. Threshold: 20% visible before trigger. Ghost/background words can slide in slightly earlier (5% visible) to feel like they're always there.
initial: opacity:0 y:28px → animate: opacity:1 y:0 · 0.5s spring · stagger: 80ms · threshold: 0.2

09 — Tone of Voice

How We Write

Honest. Confident. Calmly disruptive. The writing should feel like someone telling you something important — not shouting it.

Do
Short sentences. Let them land.
State facts confidently, without exclamation marks.
Use "cost to make" — never "discount" or "sale price."
Write like you're telling someone something they didn't know, calmly.
Trust the numbers to do the work.
Use contrast: "Most sales..." / "We..."
Let whitespace be part of the message.
Don't
Never call it a "sale" or "discount."
No exclamation marks. Ever.
Don't over-explain. Trust the reader.
No urgency language: "hurry!" "limited time!"
Don't frame as "giving back" — it's not charity.
Avoid jargon: "sustainable," "conscious consumer."
No emojis in page copy — fair elements do this visually.
